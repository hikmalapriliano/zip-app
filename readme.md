ZIP Compressor Pro 📦
ZIP Compressor Pro adalah aplikasi berbasis web sederhana namun kuat yang memungkinkan pengguna untuk mengompresi file menjadi format .zip secara langsung melalui browser. Aplikasi ini mengutamakan privasi dengan melakukan pemrosesan di sisi klien (client-side).

✨ Fitur Utama
Drag & Drop Interface: Antarmuka modern yang mendukung tarik-lepas file.

Client-Side Compression: File tidak diunggah ke server, menjaga privasi dan kecepatan data.

Custom Filename: Pengguna dapat menentukan nama file ZIP hasil kompresi.

Responsive Design: Tampilan bersih dan minimalis yang nyaman di berbagai ukuran layar.

Status Indikator: Tombol interaktif untuk memulai kompresi dan mengunduh hasil.

🛠️ Teknologi yang Digunakan
HTML5: Struktur semantik untuk antarmuka pengguna.

CSS3 (Tailwind CSS): Digunakan untuk styling modern, layout kartu, dan efek hover.

JavaScript (Vanilla): Menangani logika pemilihan file, manajemen state tombol, dan integrasi library.

JSZip (Library): Library utama yang digunakan untuk membuat dan mengelola file ZIP di browser.

🚀 Cara Menjalankan
Clone repositori ini:

Bash
git clone https://github.com/username/zip-compressor-pro.git
Buka file index.html di browser pilihan Anda.

Pastikan Anda memiliki koneksi internet jika library CSS atau JS dimuat melalui CDN.
