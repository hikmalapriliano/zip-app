# 📦 ZIP Compressor Pro

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Version](https://img.shields.io/badge/Version-1.0.0-blue)

**ZIP Compressor Pro** adalah aplikasi berbasis web yang memungkinkan pengguna untuk mengompresi file menjadi format `.zip` secara instan langsung di browser. Aplikasi ini mengutamakan privasi dengan pemrosesan penuh di sisi klien (*client-side*).

---

### ✨ Fitur Utama

* **🖱️ Drag & Drop Interface**: Antarmuka modern yang memudahkan pengguna untuk menarik dan melepas file.
* **🔒 Client-Side Compression**: File **tidak pernah** diunggah ke server. Semua proses terjadi di browser Anda, menjaga privasi 100%.
* **🏷️ Custom Filename**: Anda bebas menentukan nama file ZIP sesuai keinginan sebelum mengunduh.
* **📱 Responsive Design**: Tampilan bersih dan minimalis yang nyaman diakses dari perangkat apapun.
* **⚡ Status Indikator**: Tombol interaktif yang memberikan umpan balik visual saat proses kompresi selesai.

---

### 🛠️ Teknologi yang Digunakan

* **HTML5**: Struktur semantik yang solid.
* **CSS3 (Tailwind CSS)**: Styling modern dengan layout kartu dan efek hover yang halus.
* **JavaScript (Vanilla)**: Logika pemilihan file dan manajemen state aplikasi.
* **JSZip Library**: Mesin utama untuk pembuatan file ZIP secara dinamis.

---

### 🚀 Cara Menjalankan

Bagi Anda yang ingin mencoba menjalankan proyek ini secara lokal:

1.  **Clone** repositori ini:
    ```bash
    git clone [https://github.com/hikmalapriliano/zip-compressor-pro.git](https://github.com/hikmalapriliano/zip-compressor-pro.git)
    ```
2.  Buka file `index.html` di browser pilihan Anda.
3.  *Catatan: Pastikan Anda memiliki koneksi internet untuk memuat library Tailwind/JSZip via CDN.*
